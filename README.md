Feliz 2026!
    Este nuevo proyecto busca emular la detección de retina a partir de la apertura de los párpados.
Cuando el ojo se mantenga abierto y estable durante el tiempo requerido, el sistema confirmará la verificación correcta.
Si los párpados se cierran o no se detecta el rostro, aparecerá un mensaje de error.
