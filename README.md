# WebCam_Seguimiento_Visual
MIBU.. 1.. 2.. 3.. MIBU...
Feliz 2026!
    Este juego busca emular la detección de un ojo, y el reto principal es mantenerlo dentro de un objetivo hasta que el tiempo se acabe. Si lo logras, los anillos explotan y aparece un mensaje positivo acompañado de un sonido de victoria; si no, aparece un mensaje de error con su respectivo sonido. Durante toda la partida se escucha un beep constante que marca el ritmo, y además hay un contador de tiempo y advertencias que ayudan al jugador a mantenerse en el lugar correcto.
    Pd: Las acciones del punto y los ojos están en reflejo, lo que hace que centrar el punto sea un poco más difícil.
